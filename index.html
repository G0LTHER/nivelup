Conteúdo do Curso

Vamos começar nossa jornada! Nesta seção, você entenderá a estrutura do curso e aprenderá a configurar todas as ferramentas essenciais, como o editor de código e o navegador, para um desenvolvimento ágil.

HTML é o esqueleto de toda página web. Você aprenderá as tags mais importantes, como criar textos, listas, links e imagens, e como organizar seu conteúdo de forma semântica e profissional.

É hora de dar vida ao seu HTML! Aprenda a usar o CSS para definir cores, fontes, espaçamentos e bordas, transformando uma página simples em um design visualmente atraente.

Nesta seção, você dominará o Flexbox, a ferramenta mais poderosa do CSS para criar layouts que se adaptam a qualquer tamanho de tela, do celular ao desktop, garantindo uma experiência perfeita para todos os usuários.

Aprenda a ferramenta essencial para todo desenvolvedor. Você vai descobrir como salvar o histórico do seu código com Git e como hospedar seus projetos online no GitHub, criando seu portfólio profissional.
