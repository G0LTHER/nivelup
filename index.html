Conteúdo do Curso

O HTML é a base de toda página web o esqueleto que dá forma ao seu projeto. Nesta etapa, você aprenderá as principais tags, descobrirá como criar textos, listas, links e imagens, além de organizar o seu conteúdo, neste módulo você aprenderá:

  • A Estrutura Base do HTML e o seu primeiro projeto.
  • Você Aprenderá também a incluir Títulos, Parágrafos e Listas no seu site.
  • Será capaz de inserir Links e Imagens
  • Criar Tabelas e Formulários

É hora de dar vida ao seu HTML! Aprenda a usar o CSS para definir cores, fontes, espaçamentos e bordas, transformando uma página simples em um design visualmente atraente. Neste módulo você aprenderá:

  • Conceito Básico do CSS
  • Dar a vida ao site, utilizando Cores e Textos
  • Vai dominar o Box Model
  • Bordas e Layout Final

Nesta seção, você dominará o Flexbox, a ferramenta mais poderosa do CSS para criar layouts que se adaptam a qualquer tamanho de tela, do celular ao desktop, garantindo uma experiência perfeita para todos os usuários.

Aprenda a ferramenta essencial para todo desenvolvedor. Você vai descobrir como salvar o histórico do seu código com Git e como hospedar seus projetos online no GitHub, criando seu portfólio profissional.

DÚVIDAS SOBRE O CURSO

Perguntas e respostas frequentes sobre o curso

Sim! O NivelUp! foi criado como um projeto de extensão da UNIFENAS para ser totalmente gratuito e acessível. Nosso objetivo é repassar o conhecimento em desenvolvimento web para iniciantes.

Não é necessário! O curso é voltado para iniciantes (nível zero) e começa com os fundamentos de HTML, CSS e lógica de programação básica, preparando você do zero até a publicação do seu primeiro projeto.

Você precisará apenas de um computador (desktop ou notebook) com acesso à internet e um editor de código simples, como o VS Code (que ensinamos a instalar), e o Google Chrome ou qualquer outro navegador. Não é necessário hardware avançado.

O conteúdo central tem aproximadamente 3 horas de vídeo, mas você pode estudar no seu próprio ritmo. Recomendamos dedicar algumas horas por semana para completar o curso e o projeto prático em 2 a 4 semanas.

Atualmente, o curso não oferece um certificado formal da UNIFENAS. No entanto, o conhecimento e o projeto prático que você desenvolve são os maiores ativos para seu portfólio profissional.
